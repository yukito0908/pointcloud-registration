# 三次元点群処理

3次元点群データのレジストレーション課題に取り組んだ際の実装です。

## 概要
2つの点群の位置合わせを行うアルゴリズムを実装しました。
評価指標だけを見ると一見良さそうに見える結果でも、
全体として大きな回転ずれが残る場合があることを経験し、
局所的な数値だけでなく全体整合性を確認する重要性を学びました。

## ファイル構成
- `submissions/src/pc_reg/`: レジストレーション処理の実装
- `submissions/results/`: 出力結果
- `pyproject.toml`: 依存関係管理

## 環境
このプロジェクトは Poetry ベースで管理しています。
